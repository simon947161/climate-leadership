# Repository Relationship

## Repository Role

`climate-leadership` is intended to become a public communication site for climate leadership essays, practical explanations, civic narratives and public-facing ClimateOS ideas.

At the time of this relationship note, the repository still needs positioning cleanup and README replacement before serious use.

## Position in the Wider System

```text
ClimateOS / BuildingOS / Regional Experience
        ↓
Climate Leadership Writing
        ↓
Public communication, essays and accessible explanations
```

## Relationship to Other Repositories

- `Simon` is the personal headquarters and project index.
- `climateos-book` is the formal book publication system.
- `climateos-knowledge-garden` is the internal thinking and CRP archive.
- `BuildingOS` and `eco-agent-system` provide system ideas that may later be translated into public articles.

## Boundary

This repository should focus on:

- short essays
- climate leadership articles
- public-facing explanations
- civic and community narratives
- accessible summaries of ClimateOS / BuildingOS thinking

Do not use this repository as the main home for:

- full book manuscripts
- private working notes
- engineering task files
- system runtime architecture
- raw CRP harvests

## Operating Rule

Climate Leadership is the public newspaper column, not the research lab.

Only publish or prepare materials that are suitable for public readers.
